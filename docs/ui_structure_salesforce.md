# Структура интерфейса в стиле Salesforce

## Общие принципы дизайна

Система проектируется в стиле Salesforce CRM, но с собственными цветами и формами. Основные принципы:
- Модульная структура с четким разделением зон
- Информативность и удобство навигации
- Масштабируемость для добавления новых разделов

## Структура окна приложения

```
┌─────────────────────────────────────────────────────────────┐
│                    TopBar (верхняя панель)                   │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│ Sidebar  │      Центральный рабочий блок                   │
│ (слева)  │      (основной контент)                         │
│          │                                                  │
│          │                                                  │
├──────────┴──────────────────────────────────────────────────┤
│              BottomBar (нижняя панель)                      │
└─────────────────────────────────────────────────────────────┘
```

### Компоненты интерфейса

1. **TopBar (верхняя панель)** - как сейчас
   - Заголовок приложения
   - Кнопки действий (Создать, Экспорт, Email)

2. **Sidebar (левое меню)** - основное меню разделов
   - КП 🚀
   - **CRM 📈** (было "Торги")
   - Отгрузка 🚚
   - Клиенты 👥
   - Задачи ✅
   - AI Ассистент 🤖

3. **Центральный рабочий блок** - основной контент
   - Для раздела "CRM": отображает папки с пиктограммами (как в "Мой компьютер")
   - Для других разделов: соответствующий виджет
   - Для аналитики: отдельная страница с дашбордами (будет реализована позже)

4. **BottomBar (нижняя панель)** - ограничивающий блок
   - Статус приложения
   - Индикаторы загрузки
   - Служебная информация

## Визуальное представление папок

Папки отображаются в виде карточек с:
- Пиктограммой (иконка)
- Названием
- Кратким описанием (опционально)
- Индикатором количества элементов (опционально)

Расположение: сетка (grid layout), адаптивная под размер окна.

## Навигация

1. Клик на "CRM" в боковом меню → показываются папки
2. Клик на папку → открывается соответствующий виджет/функционал
3. Для вложенных папок (Каталог Товаров) → показывается подменю или вложенная структура

## Техническая реализация

### Новые компоненты

1. **CRMHomeWidget** - виджет с папками для раздела CRM
   - Отображает папки в виде карточек
   - Обрабатывает клики по папкам
   - Навигация к соответствующим виджетам

2. **BottomBar** - нижняя панель
   - Статус приложения
   - Индикаторы

### Изменения в MainWindow

- Добавить BottomBar снизу
- Изменить "Торги" на "CRM"
- При клике на "CRM" показывать CRMHomeWidget вместо BidsWidget напрямую
- Аналитика будет на отдельной странице с дашбордами (будущая реализация)
