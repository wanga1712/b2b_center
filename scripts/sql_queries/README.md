# SQL скрипты для работы с базой данных tender_monitor

Эта папка содержит SQL скрипты для:
- Отладки запросов к базе данных
- Оптимизации индексов
- Создания таблиц
- Проверки данных

## Структура

- **Оптимизация**: `optimize_*.sql` - скрипты для создания индексов
- **Создание таблиц**: `create_*.sql` - скрипты для создания таблиц
- **Запросы тендеров**: `new_tenders_*.sql` - запросы для получения новых тендеров
- **Отладка**: `check_*.sql`, `find_*.sql` - скрипты для отладки и поиска
